# PKBot 🤖

Простой и функциональный Telegram-бот, созданный на Python с использованием библиотеки `pyTelegramBotAPI`.

## ✨ Возможности

- 📍 Отправка приветственного сообщения
- 🔄 Обработка различных команд
- 📊 Полезные утилиты
- 🎯 И многое другое...
